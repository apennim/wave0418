# Arduino SMS 專案

這個專案是用於Arduino的SMS（短信）發送和接收系統。

## 專案描述

此專案使用Arduino開發板配合GSM/GPRS模組，實現短信的發送和接收功能。

## 功能特點

- SMS發送功能
- SMS接收功能
- 簡單易用的接口

## 硬件需求

- Arduino開發板（如：Arduino UNO、Arduino Mega等）
- GSM/GPRS模組（如：SIM800L、SIM900）
- 相關連接線材
